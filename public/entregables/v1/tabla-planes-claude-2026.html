Las 5 diferencias que importan de verdad

1. Free vs Pro: el salto fundamental

Free no tiene Claude Code, ni Cowork, ni Opus, ni Extended Thinking. Pro lo tiene TODO. Por $20/mes tienes acceso al ecosistema completo. Este es el unico upgrade realmente imprescindible.

2. Pro vs Max: solo cantidad, no features

Pro y Max tienen las MISMAS funcionalidades. La diferencia es: cuantos tokens puedes gastar (5x o 20x mas), acceso prioritario en horas pico, early access a features nuevas, y 1M de contexto automatico vs opt-in.

3. Max 5x vs Max 20x: solo volumen

Mismas features exactas. Max 20x tiene 4x mas capacidad de uso que Max 5x. Para quien quema tokens agresivamente (Fernando: "90 EUR en menos de 5 horas").

4. Team: control de equipo, no volumen

Team no da mas tokens que Pro. Aporta: SSO, admin controls, facturacion centralizada, integraciones Slack/M365, y datos que NO se usan para entrenar por defecto.

5. El contexto de 1M cambia el juego

Con 200K, Claude "olvida" despues de 2-3h de trabajo intenso. Con 1M, mantiene contexto durante 8-14h. Para proyectos grandes, esto significa no tener que re-explicar contexto constantemente.
