El ecosistema Claude: Chat, Code y Cowork

Claude no es solo un chatbot. Es un ecosistema con tres productos distintos que trabajan juntos:

ProductoQue esPara quien
Claude ChatInterfaz web y movil. Conversaciones, proyectos, artifacts.Todos: es el punto de entrada
Claude CodeAgente en terminal con acceso total a tu sistema de archivosDevelopers, consultores, power users
Claude CoworkApp de escritorio con Computer Use, Dispatch y pluginsAutomatizacion avanzada
Tip del curso

No necesitas dominar los tres desde el dia 1. Empezamos por Chat, pasamos a Code, y Cowork lo veremos como referencia avanzada.

3

Modelos: Opus, Sonnet y Haiku

Cada modelo tiene un rol distinto. La estrategia inteligente es combinarlos:

ModeloRolCuando usarlo
Opus 4.6Maximo poderPlanificar, estructurar, decisiones complejas
Sonnet 4.6Mejor balanceEjecutar tareas, desarrollo, escritura
Haiku 4.5Rapido y baratoTareas simples, clasificacion, borradores
Estrategia de cambio de modelo

Planifica con Opus (estructura y vision), ejecuta con Sonnet (el trabajo pesado), y usa Haiku para tareas rapidas. Asi optimizas tokens y dinero.

4

Ventana de contexto: el millon de tokens

La ventana de contexto es el espacio de trabajo de Claude. Cuanto mas grande, mas informacion puede procesar de una sola vez.

  • 200K tokens (Pro): ~150.000 palabras, suficiente para la mayoria de tareas
  • 1M tokens (Max): ~750.000 palabras, puedes darle un libro entero como contexto
Por que importa

Con 1M de tokens, puedes darle a Claude toda la documentacion de un proyecto, un codebase completo, o 10 PDFs de referencia y que trabaje con todo el contexto a la vez.

5

Configuracion inicial de privacidad

Antes de empezar a trabajar, hay 3 ajustes criticos:

  1. Desactiva "Ayudar a mejorar Claude": Configuracion > Privacidad. Si no, tus conversaciones se usan para entrenar el modelo.
  2. Configura la carga de herramientas: Pon "Cargar cuando sea necesario" en vez de siempre activas. Ahorra contexto.
  3. Importa tu memoria de ChatGPT (opcional): Si vienes de GPT, puedes importar tu historial.
Atencion

Si no desactivas el entrenamiento, todo lo que escribas (incluidos datos de clientes) puede usarse para mejorar el modelo. Hazlo ahora.

6

Conectores y MCPs

Claude puede conectarse a herramientas externas mediante conectores nativos y MCPs (Model Context Protocol):

  • Conectores nativos: Google Workspace, Microsoft 365, Zoho CRM, Spotify, Apify
  • MCPs: Protocolo abierto para crear conectores personalizados (Context7, Excalidraw, tu CRM)
  • Claude in Chrome: Extension beta que integra Claude en el navegador
  • Dispatch: Conecta tu movil con los archivos de tu ordenador
No necesitas todo ahora

Instala los conectores a medida que los necesites. En el modulo V4 veremos como configurar MCPs paso a paso.

7

Proyectos y Artifacts

Los Proyectos de Claude Chat son espacios de trabajo persistentes (similares a los GPTs). Puedes subir documentos como base de conocimiento, configurar instrucciones especificas y compartirlos con tu equipo.

Los Artifacts son contenido generado que puedes ver, editar y exportar:

  • Codigo ejecutable (HTML, React, Python)
  • Dashboards y visualizaciones interactivas
  • Documentos, tablas, formularios
  • Generadores de QR, calculadoras, herramientas

Prompts de esta seccion

Prompt 1: Que Claude te conozca

Prompt copiable
Quiero que me conozcas para poder trabajar juntos de forma eficiente. Hazme preguntas sobre: 1. Mi trabajo y rol profesional 2. Mis herramientas y stack tecnologico 3. Mis proyectos activos 4. Mi estilo de comunicacion preferido 5. Lo que NO quiero que hagas Hazme las preguntas una por una. Cuando termines, genera un resumen de mi perfil.

Prompt 2: Auditar tu configuracion

Prompt copiable
Actua como auditor de configuracion de Claude. Hazme preguntas para verificar que tengo bien configurado: - Privacidad (entrenamiento desactivado) - Plan adecuado a mi uso - Conectores necesarios instalados - Modelo por defecto correcto para mi tipo de trabajo Si detectas algo mal configurado, dame los pasos exactos para corregirlo.

Prompt 3: Comparar planes

Prompt copiable
Soy [describe tu rol]. Uso Claude para [tus casos de uso principales]. Mi presupuesto mensual para herramientas IA es de [cantidad]. Basandote en esto, recomiendame el plan de Claude mas adecuado. Explica que ganaria y que perderia con cada opcion.

Resumen: lo que debes recordar

  1. Claude es un ecosistema, no solo un chatbot. Chat + Code + Cowork.
  2. Plan Pro ($20) es el minimo para aprovechar el curso (acceso a Opus).
  3. Desactiva el entrenamiento inmediatamente en Configuracion > Privacidad.
  4. Opus planifica, Sonnet ejecuta, Haiku resuelve rapido — combinalos.
  5. 1M de tokens de contexto cambia las reglas del juego para proyectos complejos.
  6. Los conectores y MCPs extienden Claude a cualquier herramienta externa.

Tarea practica del modulo

  1. Verifica que tu plan es Pro o superior
  2. Ve a Configuracion > Privacidad y desactiva "Ayudar a mejorar Claude"
  3. Configura la carga de herramientas como "cuando sea necesario"
  4. Abre un Proyecto nuevo en Claude y sube un documento de trabajo real
  5. Usa el Prompt 1 de esta guia para que Claude te conozca
  6. Prueba a cambiar de modelo (Opus, Sonnet, Haiku) y observa la diferencia
